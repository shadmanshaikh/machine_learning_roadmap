:bar_chart: Statistics :

You can pick any of these and start learning at your own pace.

  • Statistical Machine Learning Book : In depth information about the various stats terms.

  • w3schools : For just the basic knowledge required to get started .

  • Basic stats concepts for newbies ! : Beginner friendly guide .

  • statistics for machine learning mini course : Mini Course on different metrics used in performing model selection and evaluation

:memo: Mathematics :

  • Mathematics for Machine Learning : Good for brushing up your high school and also uni maths with different techinques

  • Mathematics for Machine Learning by Marc Peter Deisenroth, A. Aldo Faisal, and Cheng Soon Ong. : Recommended for in depth information about the subject.

  • Article on Maths for ML by Analytics Vidhya : Good for Getting Starting.

:octocat: Python :

  • Python Basics by coursera : Good for getting started

  • complete hands on tutorial on basics of python : Hands on tutorial

  • FreeCodeCampy python : Also good for beginners

:snowflake: Standard Machine Learning Algorithms :

  • Machine Learning course by Andrew Ng : These courses are designed to be self-paced and are taught by experts in the field.

  • Machine Learning Mastery : Machine Learning Mastery is a website that provides a wide range of tutorials, courses, and books on machine learning algorithms.

  • ML by Stanford University : Stanford University offers a free online course on machine learning, which covers a wide range of topics including linear regression, logistic regression, decision trees, and clustering.
